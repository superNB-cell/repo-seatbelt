# RepoSeatbelt Demo Project

This tiny project is intentionally boring. It contains harmless fake
secret-looking values so `seatbelt doctor` can show useful warnings without
using real credentials.

Try:

```powershell
seatbelt doctor
seatbelt snapshot
seatbelt rules --agents --codex --claude --cursor
seatbelt report
```

All values in this folder are fake.
