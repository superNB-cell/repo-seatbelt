# Demo Script

Use this flow to record a simple demo GIF.

1. Open PowerShell.
2. Go to `examples/demo-project`.
3. Run `seatbelt doctor`.
4. Run `seatbelt snapshot`.
5. Run `seatbelt rules --agents --codex --claude --cursor`.
6. Run `seatbelt report`.
7. Show generated files under `.repo-seatbelt/`.

The demo project uses fake secret-looking values only.
